# TFLite models

## How to generate a model?

To generate a model, firstly setup the environment by:
```bash
source setupenv
```

Then install manually the `tensorflow` Python package:
```bash
uv run --group modelgen
```

and run the command:
```bash
generate-model
```

The model, generated by `generate-model` command, is a default one and it is
described in `scripts/model.py`. It is simple enough for the needs of running
an inference and verifying the built TF library.
